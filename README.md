# Portfolio Website

A modern, responsive portfolio website built with React and Tailwind CSS showcasing skills, projects, and contact information.

![Portfolio Preview](https://via.placeholder.com/800x400?text=Portfolio+Website+Preview)

## ✨ Features

- **Responsive Design** - Looks great on all devices from mobile to desktop
- **Dark/Light Mode** - Toggle between themes with a simple click
- **Interactive UI** - Smooth scrolling and animations
- **Project Showcase** - Highlight your best work with descriptions and links
- **Skills Section** - Organized display of technical abilities
- **Contact Form** - Easy way for visitors to reach out
- **Modern Design** - Clean and professional appearance

## 🛠️ Built With

- [React](https://reactjs.org/) - UI library
- [Vite](https://vitejs.dev/) - Build tool and development server
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Lucide React](https://lucide.dev/) - Beautiful & consistent icons
- [Class Variance Authority](https://cva.style/docs) - For UI component variants
- [TailwindCSS Animate](https://github.com/jamiebuilds/tailwindcss-animate) - Animation utilities

## 📋 Prerequisites

- Node.js (v14.0.0 or later)
- npm or yarn

## 🚀 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/marcdejesus/portfolio.git
   cd portfolio-website
   ```

2. Install dependencies
```
npm install
```

3. Start the development server
```
npm run dev
```
